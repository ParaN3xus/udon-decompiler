# Udon Decompiler 项目说明

## 项目背景

- 本仓库用于开发 **Udon 反编译器**。
- **Udon** 是 VRChat（VRC）地图使用的编程语言体系。
- 该体系包含：
  - 一套字节码/ASM 表示；
  - 对应的虚拟机（VM）执行模型。
- Udon 相关知识可以参考: local/udon-decompiler/docs/dev/udon

## 当前状态

- 旧版反编译器已使用 Python 实现。
- Python 版本代码位于：`local/udon-decompiler`。

## 当前目标

- 在本仓库中使用 **Rust** 对反编译器进行重写。
- 重写不仅是语言迁移，还包括架构层面的重构，以提升可维护性、可扩展性与性能。

## 协作原则

- 以 Rust 版本为主线推进；Python 版本作为行为参考与回归对照。
- 对核心逻辑（字节码解析、控制流恢复、语义重建）优先保证正确性，再做性能优化。
- 新增功能或修复时，尽量补充最小可复现用例，确保重构过程中行为一致。
